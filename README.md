# CREATUM Landing Page

A modern, responsive landing page for CREATUM - Agentic AI for businesses.

## 🚀 Features

- Modern responsive design
- AI Agent showcase pages
- Interactive animations
- Mobile-optimized interface
- SEO-optimized content

## 🛠 Technologies Used

- HTML5

- SVG Icons
